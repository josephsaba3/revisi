Skill that audits you brand's voice against a design.md/product.md/brand-voice.md document and ranks it based on several metrics:

- Brand Fit
- Audience Fit
- Clarity
- Human Sound
- Specificity
- Trust
- Distinctiveness

Includes contextual awareness, so that a Dentist is weighted more heavily on 'Trust' than a SaaS.

Install by pointing your agent to the repo: https://github.com/josephsaba3/brand-voice-auditor/
